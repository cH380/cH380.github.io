trim指令 ​

（1）v-trim

描述：直接使用v-trim可去掉input的前后空格

<template>
	<input class="trimInput" placeholder="请输入内容..." type="text" v-model="m1" v-trim></input>
</template>
<script setup>
    import { ref } from 'vue'
	let m1 = ref('')
</script>
<style>
    .trimInput {
        width: 200px;
		height: 30px;
        border: 1px solid #ccc;
        border-radius: 5px;
        color: #000;
    }
</style>

（2）v-trim="'all'"

描述：直接使用v-trim="'all'"可去掉input的所有空格

<template>
	<input class="trimInput" placeholder="请输入内容..." type="text" v-model="m1" v-trim="'all'"></input>
</template>
<script setup>
    import { ref } from 'vue'
	let m1 = ref('')
</script>
<style scoped>
    .trimInput {
        width: 100%;
		height: 30px;
        border: 1px solid #ccc;
        border-radius: 5px;
        padding: 0 8px;
        color: #000;
    }
</style>
