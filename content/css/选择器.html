选择器 ​

:placeholder-shown ​

设置当前显示占位符文本的输入的样式；用户未在文本框中键入任何内容

根据您的输入是否为空，应用一些动态样式非常好

html
<style>
    /* :placeholder-shown */
    input {
      border: 2px solid silver;
      padding: 3px;
      width: 100%;
    }

    .wrapper {
      position: relative;
      width: 300px;
    }


    .clear {
      position: absolute;
      top: 10px;
      right: 10px;
      background-color: #f00;
      width: 8px;
      height: 8px;
    }

    input:placeholder-shown + .clear{
      border-color: teal;
      color: #f00;
      display: none;
    }
  </style>
</head>

<body>
  <!-- :placeholder-shown -->
  <!-- 设置当前显示占位符文本的输入的样式 
  用户未在文本框中键入任何内容
根据您的输入是否为空，应用一些动态样式非常好-->

<!-- 例如可用作input的清除小按钮 -->
<div class="wrapper">
  <input placeholder="Type something here!">
  <span class="clear"></span>
</div>
    </body>
